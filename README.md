# Developing

## Prerequisites

* Node.js

## Getting started

1. Clone the repo
2. Restore packages on server and client folder ny running `npm install`

## Running

Server:

```
> npm run start
```

Client:

```
> npm run serve
```

Navigate to http://localhost:8080
